# Backend - NestJS API

## Install
### 1. Clone repository
```
git clone https://github.com/RomeoLeyn/random-users-app.git
cd ./server
```

### 2. Install dependencies
```
npm install
```

### 3. Project launch
```
npm start
npm run start:dev
```

## API Endpoints
### Users
- `POST /users` - user add
- `GET /users/:page` - get users
- `DEL /users/:id` - user delete
